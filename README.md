# NewswireZEX Website

A fresh static website inspired by a press release distribution service. The site includes a homepage,
service package cards, industry coverage, partner callout, resources, FAQ, and a demo contact form.

## Files

- `index.html` - page structure and content
- `styles.css` - responsive visual design
- `script.js` - mobile menu and demo form behavior

## Run Locally

Open `index.html` in a browser, or serve the folder with any static web server.
